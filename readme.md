wordSwap
=============

* A simple jQuery plugin that allows transitions between snipets of text.
* You can pass single words for a one-time thing, or arrays that will loop.



Usage:

$('#wordSwaparea').wordSwap({
	wordArray:['Progress',
				'Potential',
				'Perseverance',
				'Profit'
				]
});
